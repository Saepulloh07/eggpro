DAFTAR BUG (QA FINDINGS)
========================

1. [MINOR] Input Angka Negatif
   - Lokasi: Input klasifikasi telur (Production.tsx)
   - Deskripsi: Pengguna masih bisa memasukkan angka negatif melalui keyboard di beberapa field input. Meskipun backend memvalidasi, frontend harus memblokir sejak awal.
   - Dampak: Inkonsistensi data jika validasi backend gagal.
   - Rekomendasi: Gunakan `min="0"` dan validasi `Math.max(0, val)` pada handler onChange.

2. [SEDANG] Race Condition pada Sinkronisasi Offline (Edge Case)
   - Lokasi: syncUtils.ts / LocalStorage sync
   - Deskripsi: Jika dua user melakukan update data yang sama saat offline, data terakhir yang online akan menimpa data sebelumnya (Last Win).
   - Dampak: Potensi kehilangan data audit.
   - Rekomendasi: Implementasikan conflict resolution berbasis timestamp atau versioning pada setiap baris data.

3. [MINOR] Format Tanggal Excel
   - Lokasi: Finance.tsx (handleExportExcel)
   - Deskripsi: Di beberapa versi Excel, format tanggal ISO dari JavaScript terkadang tidak terbaca sebagai object Date Excel.
   - Dampak: User kesulitan melakukan filter tanggal di Excel.
   - Rekomendasi: Gunakan `new Date(row.date)` secara eksplisit saat menulis ke cell ExcelJS.
